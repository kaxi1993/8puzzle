# 8 Puzzle

React web app which solves **8 Puzzle** problem using A* algorithm

## Visualization
![alt text](http://kaxi1993.github.io/images/projects/8puzzle/game.gif)

## Installation
```bash
git clone https://github.com/kaxi1993/8puzzle.git
```

```bash
cd 8puzzle
```

```bash
npm i
```

```bash
npm start
```
